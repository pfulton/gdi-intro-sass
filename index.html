What is Sass?

  • "Sass" stands for "Syntactically awesome style sheets".
  • An extension of CSS that adds to the language.
  • One of several CSS "preprocessors".

What's a CSS Preprocessor?

A scripting language that extends CSS and gets compiled into regular CSS syntax.

Compiled?

Sass ➡️ compile ➡️ CSS ➡️ browser

We'll be looking at this in depth as we go through the workshop today.

Other CSS Preprocessors

  • LESS
  • Stylus

Why Use a CSS Preprocessor?

  • Cleaner code with reusable pieces
  • Shareable snippets and libraries
  • Cuts down on repeated code
  • Programatic
  • Fun!

Mixins

A mixin is a block of code that lets us group CSS declarations we may reuse throughout our site.

Mixins

						
							.btn-default {
								background-color: pink;
								border-radius: 4px;
								color: white;
								display: inline-block;
								font-size: 1.0em;
								text-align: center;
							}

Mixins

						
							$button-text-color: white;

							@mixin basic-button {
								background-color: pink;
								border-radius: 4px;
								color: $button-text-color;
								display: inline-block;
								font-size: 1.0em;
								text-align: center;
							}
						
					
						
							.btn-default {
								@include basic-button;
							}

Mixins - Advanced

Mixins can get complex! You can pass arguments to them.

						
							@mixin gridset($type) {
								@if $type == 'flex' {
									display: flex;
								} @else {
									display: block;
								}
							}
						
					
						
							.gridset {
								@include gridset(flex);
							}
						
					
						
							.gridset {
								display: flex;
							}

Debugging!

The mixin I just showed has a subtle error in it.

						
							.gridset {
								@include gridset;
							}
						
					
						
							Mixin gridset is missing argument $type.

Mixin Refactoring

						
							@mixin gridset($type: block) {
								@if $type == 'flex' {
									display: flex;
								} @else {
									display: block;
								}
							}
						
					
						
							.gridset {
								@include gridset;
							}
						
					
						
							.gridset {
								display: block;
							}
